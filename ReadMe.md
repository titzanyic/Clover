## 安装步骤

### 设置 BIOS

##### BIOS：

- 【Fast Boot】  **Disabled**
- 【GSM Support】 **Disabled** （推荐关闭，我测试不关闭不影响）

##### Peripherals：

- 【USB Configuration > XHCI Hand-off】  **Enabled**  （重要！！！）

##### Chipset

- VT-d Disabled
